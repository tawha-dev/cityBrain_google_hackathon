# frontend/

Client applications for the CityBrain command center.

| Subfolder | Purpose |
|-----------|---------|
| `mobile/` | Expo React Native — primary hackathon deliverable |
| `web/` | Browser command center (Expo web or Vite) |

**Imports:** `@citybrain/shared` only (types, events, constants).  
**Must not import:** `backend`, `ai-agents`, `simulation-engine`.

See [docs/PROJECT_STRUCTURE.md](../docs/PROJECT_STRUCTURE.md).
